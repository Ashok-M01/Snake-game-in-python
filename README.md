# Snake Game in Python

A classic Snake Game built with Python and Pygame, designed to work well on Android devices using on-screen touch controls.

## Features

- 🐍 Smooth snake movement
- 🍎 Random food generation
- 📈 Score counter
- ⚡ Speed increases as the score grows
- 📱 On-screen touch controls (UP, DOWN, LEFT, RIGHT)
- 💀 Game Over screen
- 🔄 Tap anywhere to restart

## Requirements

- Python 3.x
- Pygame

Install Pygame:

```bash
pip install pygame
```

## Run

```bash
python snakefinal.py
```
